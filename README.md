# AgriFert Frontend

A React-based web application for agricultural fertilizer management and recommendations.

## Features

- Modern React 18 application with Material-UI components
- Responsive design with Bootstrap integration
- Data visualization with Chart.js and Recharts
- PDF generation and export capabilities
- User authentication and management
- Interactive charts and analytics
- Email integration with EmailJS

## Tech Stack

- **Frontend Framework**: React 18
- **UI Libraries**: Material-UI, Ant Design, Bootstrap
- **State Management**: Redux Toolkit
- **Routing**: React Router DOM
- **Charts**: Chart.js, Recharts
- **PDF Generation**: jsPDF, React-PDF
- **HTTP Client**: Axios
- **Styling**: Styled Components, Emotion

## Prerequisites

- Node.js (version 14 or higher)
- npm or yarn package manager

## Installation

1. Clone the repository:
```bash
git clone 
cd frontend
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

The application will open at `http://localhost:3000`

## Available Scripts

- `npm start` - Runs the app in development mode
- `npm build` - Builds the app for production
- `npm test` - Launches the test runner
- `npm run eject` - Ejects from Create React App (one-way operation)
